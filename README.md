# Sistem Perpustakaan Digital Publik SYR

## Deskripsi
Repositori ini berisi implementasi **Sistem Perpustakaan Digital Publik SYR**, yaitu aplikasi web perpustakaan digital yang dirancang untuk memudahkan pengelolaan serta akses koleksi perpustakaan secara daring.

Aplikasi ini dikembangkan sebagai proyek aplikasi web dan telah **dideploy pada platform Microsoft Azure**, sehingga dapat diakses secara publik melalui internet.
[Live Demo](https://web-perpus-syr-dyg3cxh6euexckd6.southeastasia-01.azurewebsites.net/)

## Tujuan Proyek
Proyek ini bertujuan untuk menunjukkan pemahaman konsep:
- Pengembangan aplikasi web
- Integrasi frontend dan backend
- Pengelolaan data perpustakaan secara digital
- Implementasi deployment aplikasi pada cloud (Microsoft Azure)

## Struktur Proyek
```text
.
├── static/           # File CSS, JavaScript, dan aset statis
├── templates/        # Template tampilan web
├── app.py            # File utama aplikasi
├── database/         # Konfigurasi atau skrip database
└── README.md         # Dokumentasi proyek
```

## Fitur
- Pengelolaan data buku perpustakaan
- Penampilan koleksi buku secara digital
- Akses perpustakaan berbasis web
- Antarmuka sederhana dan mudah digunakan
- Akses publik melalui cloud

## Komponen
```text
Bagian	Teknologi
```

## Deployment
Aplikasi ini telah dideploy menggunakan layanan Microsoft Azure Web App dengan detail:
- Platform: Microsoft Azure
- Region: Southeast Asia
- Mode akses: Public Web Application

Pengembangan lebih lanjut dapat dilakukan sesuai kebutuhan fitur dan skalabilitas sistem.
